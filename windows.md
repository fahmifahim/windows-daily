#### Find Windows 10 Serial Number
```bash
# command
wmic csproduct get identifyingnumber

# result
IdentifyingNumber     Name
JPA3039FWR              HP EliteBook 2570p

-or-
# command 
wmic bios get serialnumber

# result
Serial Number
JPA3039FWR

```
